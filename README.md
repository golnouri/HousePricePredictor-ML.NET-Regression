# House Price Predictor (ML.NET Console App)
A simple console application written in **C# (.NET)** that demonstrates how to use **ML.NET** for a basic **Regression** task.

The model predicts the approximate **house price** based on:
- House size (meters)
- Number of rooms

## Features
- Simple regression model using ML.NET
- Training with sample data
- Console input for predictions
- Clean and easy-to-understand structure for learning purposes

## Technologies Used
- .NET
- C#
- ML.NET

## How It Works
1. The application trains a regression model using sample housing data.
2. The user enters:
   - House size
   - Number of rooms
3. The model predicts an estimated price.
